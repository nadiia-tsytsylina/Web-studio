# Web-studio

This is a digital marketing company website with responsive layout.

The web-site contains 3 pages:

- Home page - the page with general information about the company, its advantages, company team and customers with ability to leave user's contacts;
- Portfolio page - the page with company's works;
- Contacts page.

## Technology stack

- HTML5
- CSS
- JavaScript
